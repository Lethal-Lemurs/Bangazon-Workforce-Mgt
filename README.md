# Bangazon-Workforce-Mgt

## Linux
In your config.json file add your current user and the password you use to access that user with psql. 

>Example
```javascript
"development": {
    "username": "vm",
    "password": "pass",
    "database": "bangazon_workforce",
    "host": "127.0.0.1",
    "dialect": "postgres"
    }
```
